# Core Framework

The core framework contains reusable orchestration and cognitive infrastructure for AISystemsEdge.

This layer is intended to remain domain-independent whenever possible.

Core responsibilities include:

- workflow orchestration
- graph execution
- memory persistence
- checkpoint recovery
- provider abstraction
- runtime coordination
- shared schemas
- state propagation

The purpose of the core layer is to provide a reusable cognitive execution engine capable of supporting multiple enterprise and operational domains.
